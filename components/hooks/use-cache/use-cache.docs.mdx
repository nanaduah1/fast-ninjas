## A React Hook create a cache

### Component usage

```js
import { useCache } from "./use-cache";

const cache = useCache();
const getLatest = () => {
  cache.getOrSet("my-key", () => {
    console.log("Loading from source");
    return 4000;
  });
};

<button onClick={getLatest}>increment</button>;
```
